# 05 CLI Agent (System Command Agent)

This directory contains an advanced AI agent capable of writing and executing actual terminal and shell commands on your local machine to fulfill user requests.

⚠️ **WARNING: SECURITY RISK** ⚠️
> This script uses `os.system(cmd)` to run commands generated by an AI directly on your host machine. **Do not run this script with elevated (Admin/root) privileges**, and be extremely cautious about the prompts you provide. The AI could accidentally execute destructive commands (like deleting files). It is highly recommended to run this in a sandboxed environment, such as a Docker container or a Virtual Machine.

## Overview

Unlike previous agents that fetched data from read-only APIs, this script demonstrates an agent with write/execute permissions on your local operating system. It uses OpenAI's **Structured Outputs** (via Pydantic) to guarantee that the agent follows a strict Chain of Thought schema.

## 🌟 Capabilities in Action: The Todo App

To understand the power of this CLI agent, look at the `todo` directory in this repository. **The entire Todo application (HTML, CSS, and JavaScript) was autonomously generated and written to the filesystem by this exact agent.** Because it can execute commands, you can prompt it to "create a folder named 'todo' and build a functional to-do list web app inside it." The agent will plan the files needed and use standard terminal commands (like `echo` or file redirection) to write the code directly to your disk.

## How It Works

* **`main.py`**: The core script. It establishes a chat loop and instructs the `gpt-4o` model to think step-by-step using a defined JSON format (`START`, `PLAN`, `TOOL`, `OBSERVE`, `OUTPUT`). 
* **The Tool (`run_command`)**: The agent is provided with a single tool called `run_command(cmd: str)`. When the agent decides it needs to perform a system action (like checking the weather, reading a file, or creating a folder), it passes the command to this tool, which executes it via Python's `os` module and returns the output.
* **Structured Parsing**: Instead of manually parsing JSON strings, this script uses OpenAI's `.parse()` method alongside a Pydantic `BaseModel` (`MyOutputFormat`) to strictly enforce the shape of the LLM's response.
