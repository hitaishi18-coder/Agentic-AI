# 08 - Vision Models: Seeing through AI 👁️

This module introduces **Vision Models** (also known as Multimodal LLMs), which go beyond text to "see" and understand images.

---

## 📘 Theory: What are Multimodal Models?

Traditional LLMs only process text. **Multimodal models** can process text plus other types of data, like images or videos.

### Key Concepts:
1.  **Vision Prompting**: Instead of just sending a "text" content, we send a **list** of contents containing both "text" and "image_url".
2.  **Multimodal LLMs**: Models like `GPT-4o`, `Claude 3.5 Sonnet`, or `Llama-3.2 Vision` that have been trained on both text and visual datasets.
3.  **Image URL vs. Base64**: You can send images to an AI either via a direct **URL** or by encoding the image file itself into a **Base64 string**.

---

We use the `openai` library to interact with Vision models. Even if the model is from Meta (Llama), many providers provide an "OpenAI-compatible" interface.

## 💻 Code Explanation (Simplified)

### 1. Creating the Client
As always, we load our environment variables and initialize the client.
```python
load_dotenv()
client = OpenAI()
```

### 2. The Multimodal Request
In a vision request, the `content` field is a **list of dictionaries**.
- The first dictionary is of type `"text"`.
- The second is of type `"image_url"`.

```python
response = client.chat.completions.create(
    model="llama-3.2-11b-vision-instruct", # The Vision model
    messages=[
        {
            "role": "user",
            "content": [
                {"type": "text", "text": "What is in this image?"},
                {"type": "image_url", "image_url": {"url": "https://link-to-image.jpg"}}
            ]
        }
    ]
)
```

## 📜 Full Code Listing: `main.py`

```python
from dotenv import load_dotenv
from openai import OpenAI

load_dotenv()
client = OpenAI()

# 1. Provide an image URL and a text prompt
response = client.chat.completions.create(
    model="meta-llama/llama-3.2-11b-vision-instruct", # Vision model
    max_tokens=100,
    messages=[
        {
            "role": "user",
            "content": [
                {"type": "text", "text": "Generate a caption for this image"},
                {
                    "type": "image_url",
                    "image_url": {
                        "url": "https://images.pexels.com/photos/18105/pexels-photo.jpg"
                    }
                }
            ]
        }
    ]
)

# 2. Print the description of the image generated by the AI
print(response.choices[0].message.content)
```
